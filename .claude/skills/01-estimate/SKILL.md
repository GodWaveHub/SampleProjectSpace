# 見積作成時の指示
- 見積作成時は必ず見積作成のルールに従って、見積を作成してください。
- フォーマットは./assetsを参照してください。

## 見積作成のルール
- 見積書は、01_Estimate フォルダに作成してください。
- 見積は、プロジェクトの要件に基づいて正確かつ、かつ実行可能なものとしてください。
- 見積は、プロジェクトのスコープ、リソース、時間ラインを考慮して作成してください。
- 見積は、チームの経験と過去のプロジェクトデータを参考にしてください。 
- PARAMETERS.mdの係数・生産性・単価を参照し 規模(LOC/FP)算出 → 工数換算 → コスト積算 を一貫管理  
- 使用したパラメータ（人月単価・生産性・補正係数・稼働率等）を PARAMETERS.md から引用し明記（表形式推奨）  
- 要求変更・前提条件変更時に再計算し版管理（例: v1.0 → v1.1）  
- 見積項目と WBS タスクID を対応付け、差異分析を Progress にフィードバック
